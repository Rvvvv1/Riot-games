# BiN Lee Portfolio 🎨

个人作品集网站，展示技术栈与项目经验。

## 技术栈

| 分类 | 技术 |
|------|------|
| **框架** | React 19 + TypeScript |
| **构建工具** | Vite 8 |
| **样式** | TailwindCSS 4 |
| **动画** | GSAP |
| **图标** | Lucide React |
| **代码质量** | ESLint + Prettier |
| **包管理器** | Bun |

## 特性

- Vite HMR
- GSAP
- TailwindCSS
- TypeScript
- React Compiler

### 安装依赖
bun add

### 快速开始
bun dev

### 环境要求

- Node.js/Bun
